# retro-shooter

A small collection of single-file browser games. No build step, no dependencies — just open the HTML file in a browser.

## Games

### `shooter.html` — Retro Shooter
A 2D top-down arena shooter with procedural pixel-art sprites.

- **Move:** Arrow keys / WASD
- **Aim:** Mouse
- **Shoot:** Left click (hold to autofire)

Three levels of progression, four enemy types (grunt, runner, shooter, boss), wave-based spawning, screen shake, particle effects.

### `tictactoe.html` — Tic Tac Toe
Two-player hot-seat tic-tac-toe. Winning row highlights green.

## Running

Open either file directly in a browser:

```sh
open shooter.html
open tictactoe.html
```
